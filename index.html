#2: HTML - Details & Summary

What is a <details> element?

Each <details> element denotes a single collapsible area. Any content insdie that block will be hidden by default unitl the Details heading is clicked.

Adding an 'open' attribute will expand the block. This can also be triggered through JavaScript as a way to reveal certain info to the user

Multiple <details> elements can also be nested without issue. See below.

What is a <summary> element?

Adding a <summary> to the top of the block will replace the default heading to the contents of that element. The heading then becomes interactive.


Which browser supports both <details> and <summary> elements?

Both of these elements are available in all browsers apart from Edge and IE.
